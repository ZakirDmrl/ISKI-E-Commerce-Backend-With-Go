# E-commerce Backend (Go)

Go ile yazılmış e-commerce backend API'si.

## Özellikler

- JWT Authentication
- Product Management
- Shopping Cart
- Comments System
- Admin Panel
- File Upload (Avatar)

## Kurulum

1. Repository'yi clone edin
2. `.env.example`'ı `.env` olarak kopyalayın
3. Environment variables'ları düzenleyin
4. Dependencies'leri yükleyin:
   ```bash
   go mod tidy
